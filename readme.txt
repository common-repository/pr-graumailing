Plugin de integração entre Contact Form 7 e Graumailing.

== Description ==

Este é um simples plugin que auxilia na integração entre os formulários de contato gerados através
do Contact Form 7 e o sistema de envio de emails Graumailing de maneira rápida, segura e confiável.

== Installation ==

1. Envie o arquivo `pr-graumailing.zip` para o diretório `/wp-content/plugins/`
1. Ative o plugin através do menu Plugins do WordPress
1. Crie as integrações através do sub-menu PR Graumailing dentro do menu do Contact Form 7
